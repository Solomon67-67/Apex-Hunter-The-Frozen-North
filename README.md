Apex Hunter: Frozen NorthA high-stakes 3D survival simulation built with Three.js. Stranded in the Siberian wilderness, you must reach the extraction flare while managing your stamina and avoiding an unseen predator tracking you through the snow.🚀 Features3D Environment: Fully realized forest with dynamic fog and lighting.First-Person Survival: Physics-based movement including sprinting and stamina management.Dynamic AI: A predator that stalks the player based on proximity.Atmospheric Visuals: Custom shaders and lighting to simulate the "Frozen North."🎮 How to PlayOpen index.html (or apexhunter.html) in any modern web browser.Click "Begin Survival" to lock your mouse and start.Controls:WASD / Arrows: MoveMouse: Look aroundShift: Sprint (consumes stamina)Goal: Locate the glowing blue vertical flare to be extracted.🛠️ TechnologyThree.js: 3D Rendering EngineTailwind CSS: HUD and UI StylingJavaScript (ES6): Game logic and AI📜 LicenseThis project is licensed under the MIT License - see the LICENSE file for details.
